# Phase 2 — Economy and Planet Cost

## Goal
Make harvesting a strategic risk/reward system.

## Deliverables
- Multiple shard field types.
- Field depletion/regrowth.
- Multiple harvesters.
- Processor queues/deposit blocking.
- Silo capacity.
- Red shard volatility.
- Local agitation grid.
- First shard bloom event.

## Acceptance Criteria
- Blue and red fields behave differently.
- Red fields generate more Aether and more agitation.
- Fields deplete.
- Silo cap works.
- Shard Bloom warning and event can occur.
