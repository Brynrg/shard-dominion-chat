# Phase 1 — Terrain, Fog, Pathfinding

## Goal
Make the map matter. Add terrain movement rules, obstacles, fog/shroud, and A* pathfinding.

## Deliverables
- Terrain config.
- Movement costs.
- Blocked ridges.
- Deep crust worm-risk terrain.
- A* pathfinding.
- Fog visible/explored masks.
- Vision radius per unit.
- Minimap placeholder.

## Acceptance Criteria
- Units path around blocked tiles.
- Deep crust slows movement.
- Fog hides unexplored map.
- Explored map remains dimly visible.
- Vision follows units.
- Debug overlay shows movement cost and worm risk.
