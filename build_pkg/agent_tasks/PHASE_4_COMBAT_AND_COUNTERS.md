# Phase 4 — Combat and Counters

## Goal
Add readable combat with basic unit roles.

## Deliverables
- Weapons config.
- Damage/armor multiplier table.
- Infantry, rover, tank, missile carrier, turret.
- Attack order.
- Attack-move.
- Projectile/effect placeholders.
- Death/destruction.

## Acceptance Criteria
- Units can attack enemies.
- Counters are noticeable.
- Turrets defend base.
- Units die and are removed cleanly.
- Harvester raiding is possible.
