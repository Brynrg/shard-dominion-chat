# Phase 3 — Base Building, Power, Lattice

## Goal
Implement the core base-building identity.

## Deliverables
- MCV deployment.
- Build sidebar prototype.
- Lattice placement.
- Building placement footprints.
- Raw-ground placement and decay.
- Power generation/use.
- Power deficit modifiers.
- Building repair.

## Acceptance Criteria
- Player can place lattice.
- Player can place Power Node, Processor, Barracks.
- Raw-ground buildings decay.
- Power deficit slows production.
- Building placement validates footprint and build radius.
