# Phase 5 — UI, Commands, Repair

## Goal
Make the prototype feel like a proper RTS.

## Deliverables
- Sidebar.
- Selection panel.
- Build queue display.
- Minimap.
- Control groups.
- Rally points.
- Stop/guard/patrol/attack-move.
- Repair mode.
- Alerts.

## Acceptance Criteria
- Player can control army comfortably.
- UI clearly shows resources, power, agitation, selection, queues.
- Minimap supports click-to-camera.
