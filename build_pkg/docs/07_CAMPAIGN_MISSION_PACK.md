# Shard Dominion — Campaign Mission Pack

## Mission 1: First Lattice — Vanguard
Tutorial for movement, lattice, power, harvesting, scouting, basic defense.

Objectives:
1. Deploy MCV.
2. Build 6 Basic Lattice tiles.
3. Build Power Node.
4. Build Processor.
5. Harvest 2,000 Aether.
6. Scout Old Relay Tower.
7. Survive Forge raid.
Optional: Lure Titan juvenile into raiders.

## Mission 2: Claim the Red Field — Forge
Tutorial for red shards, overproduction, armor, volatility.

Objectives:
1. Capture red shard basin.
2. Build two Processors.
3. Use Foundry Overload.
4. Destroy Vanguard outpost.
Optional: Keep global agitation below 65.

## Mission 3: Voices in the Dust — Phantom
Tutorial for stealth, sabotage, non-destruction objectives.

Objectives:
1. Infiltrate relay zone.
2. Disable two Power Nodes.
3. Capture data without destroying relay.
4. Escape before Glass Storm.
Optional: Complete without being detected.

## Mission 4: The Worm Road — Vanguard
Escort and terrain-risk mission.

Objectives:
1. Escort three Harvesters across deep crust.
2. Build Survey Towers.
3. Use thumpers to redirect worm sign.
4. Establish expansion.
Optional: Lose no Harvesters.

## Mission 5: Furnace Line — Forge
Siege base under bloom pressure.

Objectives:
1. Establish forward lattice.
2. Build Kiln Siege Rigs.
3. Crack enemy fortress.
4. Survive shard bloom chain.
Optional: Destroy fortress with worm collapse.

## Mission 6: Borrowed Machines — Phantom
Hijack mission.

Objectives:
1. Hijack enemy convoy.
2. Use stolen vehicles to breach base.
3. Destroy Command Nexus.
Optional: Keep original Phantom force alive.

## Mission 7: Stabilization Protocol — Vanguard
Moral reveal.

Objectives:
1. Activate three Stabilizer Towers.
2. Defend towers.
3. Discover that stabilization suppresses colony memory signal.
Choice: Continue or partially disable network.

## Mission 8: The Sleeping Engine — Forge
Buried machine mission.

Objectives:
1. Excavate ancient mining engine.
2. Power it with Black Shards.
3. Survive Titan emergence.
Choice: Use engine or destroy it.

## Mission 9: Dominion Shard — Phantom
Final revelation.

Objectives:
1. Reach Black Shard core.
2. Disable rival command systems.
3. Decode colony memory.
4. Choose extraction, stabilization, convergence, or accord.
