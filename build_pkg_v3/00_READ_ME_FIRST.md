# Shard Dominion — Complete Execution Package

This package is the authoritative build plan for converting the existing `Brynrg/shard-dominion-chat`
repository into a complete, original, browser-based real-time strategy game comparable in scope,
playability, and strategic depth to a late-1990s commercial RTS.

It is not a request to restart the codebase.

The builder team must:

1. repair the current runtime;
2. complete one production-quality vertical slice;
3. expand into a full faction-complete combat game;
4. add skirmish AI;
5. complete the nine-mission campaign;
6. replace temporary assets;
7. pass the release gates.
