# T03 — Real Economy Credits

## Problem
Credits currently derive from carried cargo rather than deposited cargo.

## Tasks
- Add player resource state
- Add shard field amount per tile/deposit
- Harvester cargo fills over time
- On reaching processor, cargo converts to credits
- Field amount decreases
- Empty fields stop yielding
- HUD shows credits and cargo correctly

## Acceptance
- credits increase only on deposit
- cargo bar fills while harvesting
- cargo clears on deposit
- fields can be depleted
