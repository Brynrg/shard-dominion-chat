# 04 — First Lattice Vertical Slice

This is the first complete playable proof that the game works.

## Mission title

**First Lattice**

## Player faction

Vanguard Enclave

## Narrative premise

The Vanguard survey ship *Resolute Dawn* lands on Aether Prime after detecting a repeating distress signal from a dead colony. The first landing zone is unstable. The player must deploy a base, harvest enough Aether to power a relay, and survive the planet's first reaction.

## Player starts with

- 1 Mobile Command Vehicle
- 1 Harvester
- 2 Surveyors
- 2 Rifle Teams
- 500 credits

## Map ingredients

- safe blue shard field near start
- red volatile shard field farther away
- ridge line creating a natural choke
- deep crust lane with worm risk
- old relay tower
- small Forge raiding party scripted event
- one shard bloom event

## Required objectives

1. Deploy MCV into Command Core.
2. Place 8 Anchor Lattice tiles.
3. Build Power Node.
4. Build Processor.
5. Harvest and deposit 2,000 Aether.
6. Scout Old Relay Tower.
7. Survive first Shard Bloom.
8. Defeat or evade Forge raiders.

## Optional objective

Lure a Titan Worm sign event or crust tremor into damaging Forge raiders.

## Mechanics proven

- unit selection
- right-click movement
- pathfinding
- fog of war
- minimap
- harvesting/deposit
- credits
- construction
- lattice placement
- power
- basic combat
- planet agitation
- first environmental event
- story briefing
- objective tracker

## Acceptance criteria

The vertical slice passes only if:
- a new player can understand what to do without reading docs
- harvester economy works
- building placement works
- power has visible consequence
- planet agitation changes based on actions
- at least one planet event changes the mission state
- the player can win and lose
